#  SafeCircle

SafeCircle is a Smart Emergency Web Application designed for student safety.

##  Features

-  One-Tap SOS with Live Location
-  Emergency Audio Recording
-  Fake Incoming Call Mode
-  Google Maps Link Sharing

##  Tech Stack

- HTML
- CSS
- JavaScript
- Geolocation API
- MediaRecorder API

##  Purpose

To provide instant emergency assistance through a browser-based solution without requiring app installation.

##  Future Scope

-Real Time Emergency Alerts
- Secure Data Storage
- Live Location Monitoring
- AI based Threat Detection
